# React Landing Page

Лендинг-страница на React.

🌐 **Демо**: https://sumaya20011.github.io/React-lending-page/

## Структура проекта

```
src/
├── components/
│   ├── Header.js       # Шапка сайта с навигацией
│   ├── Header.css
│   ├── SpaceAgency.js  # Секция «Космическое агентство»
│   ├── SpaceAgency.css
│   ├── Photos.js       # Слайдер фотографий
│   ├── Photos.css
│   ├── Footer.js       # Подвал сайта
│   └── Footer.css
├── App.js              # Главный компонент
└── index.js            # Точка входа
```

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Откройте http://localhost:3000 в браузере.

## Стек

- React 18
- CSS
